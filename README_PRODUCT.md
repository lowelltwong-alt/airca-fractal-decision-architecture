# AIRCA Core

AIRCA Core is the practical, developer-facing layer of the AIRCA Fractal Decision Architecture repository.

## One-line pitch

AIRCA helps humans and AI systems make structured decisions that are explainable, machine-readable, and easy to store in GitHub.

## What it does

AIRCA Core provides:

- a minimal Python decision engine
- machine-readable decision artifacts
- markdown decision records
- a CLI for generating demo artifacts
- example use cases for adoption

## Why it exists

The larger repository develops a broad theory of decision capacity, recursive governance, and fractal structure.

AIRCA Core makes those ideas usable.

## Try it

```bash
pip install -e .
airca demo --output-dir output
```

## Key files

- `pyproject.toml`
- `airca/`
- `schemas/airca-decision.schema.json`
- `examples/python/`

## Best first use case

Use AIRCA Core for AI-assisted architecture and governance decisions where you want:

- accountability
