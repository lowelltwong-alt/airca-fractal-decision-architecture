# Game Theory and Mechanism Design

AIRCA should use game theory selectively and practically.

## Where game theory matters

AIRCA can benefit from game-theoretic thinking when decisions involve:

- multiple actors
- conflicting incentives
- incomplete information
- escalation pathways
- negotiation and approval structures

## Mechanism design relevance

AIRCA is not only about making choices.

It is also about designing the rules of the choice process itself.

That includes:

- who must review
- how options are disclosed
- when escalation is required
- how dissent is recorded
- what evidence is required before commitment

## Practical AIRCA use

Game theory should help AIRCA answer:

- when actors may hide information
- when incentives are misaligned
- when escalation should be automatic
- when approvals should be distributed rather than centralized
