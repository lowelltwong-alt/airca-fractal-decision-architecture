# Framework Coherence and Decision Layer

This node places the coherence and scoring layer of the framework into the fractal decision-making structure.

## Purpose

While ontology defines structure, this layer defines behavior:

- coherence scoring
- contradiction penalties
- dependency penalties
- reinforcement bonuses
- propagation logic
- misalignment measurement

## Core idea

The system is not only a graph. It is:

- graph structure
- plus state vector of active commitments
- plus scoring rules

Changes to one node should propagate through dependent and conflicting nodes.

## Alignment with AIRCA

This layer connects directly to AIRCA:

- Architect defines constraint structures and scoring rules
- Inform surfaces graph and state
- Rank uses scoring outputs
- Commit sets activation state of views
- Act executes downstream implications

## Outputs

- local tension
- cluster tension
- global coherence score
- misalignment vectors

This turns ontology into a decision-aware system rather than a static map.
