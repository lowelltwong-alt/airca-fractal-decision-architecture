# Integrations Lineage

This note records the lineage, attribution boundaries, and conceptual sources for the integrations node.

## Purpose

The purpose of this file is to distinguish among:

- collaborative inputs
- external intellectual influences
- schema-derived architectural structure
- original synthesis developed in this repository

This matters because the integrations node is where many of the repository’s ideas are connected into a broader architecture, and that architecture includes both inherited and original elements.

## AIRCA-related lineage

The integrations node draws in part on AIRCA as a decision framework that emerged through a collaborative process between Alex A. and Lowell T. Wong.

Within this repository, AIRCA is extended through:

- fractal taxonomy
- fractal ontology
- fractal decision-making
- machine-readable artifact design
- recursive governance
- UAIRCA and LAIRCA as higher-order extensions

These extensions are primarily developed here by Lowell T. Wong.

## Schema-derived structure

The integrations node also draws heavily on the fractal decision artifact schema and template work that already established:

- recursive nodes
- stable field families
- lifecycle logic
- inheritance logic
- interpretation rules
- artifact tiering and class naming
- safety rules around blank and missing values

These schema materials provide much of the operational backbone for the machine-readable side of the integrations architecture.

## External intellectual lineage

This node is also informed by adjacent external work, especially in relation to:

- decision intelligence
- Bayesian decision theory
- control systems and cybernetics
- graph theory and knowledge graphs
- hierarchical and multi-scale systems
- recursive data structures
- constraint optimization
- latent structure and decision landscapes

This repository does not claim those broader domains as original. It uses them as neighboring conceptual frameworks that help explain or strengthen the architecture being developed here.

## Theological and higher-order extension

- Ultimate Frame

These higher-order extensions are treated here as part of Lowell T. Wong’s repository-specific synthesis.

## Attribution boundary

This node distinguishes among:

- AIRCA as collaboratively emergent
- schema logic as an existing structural foundation
- external decision/intelligence/technical concepts as intellectual lineage
- the integrated architecture developed here as a repository-specific synthesis by Lowell T. Wong

## Working thesis

The integrations node is where this repository becomes more than a collection of related ideas.

It is where collaborative AIRCA lineage, fractal schema logic, external conceptual influences, and Lowell T. Wong’s original synthesis are brought together into a more coherent architecture.
