# Python Examples

This folder contains practical starting points for AIRCA.

## Included example

### Task prioritization demo

This example shows how AIRCA can be used to:

- define a decision question
- compare structured options
- rank choices using explicit criteria
- commit a choice
- generate machine-readable and markdown artifacts

## Run it

From the repository root:

```bash
python examples/python/task_prioritization_demo.py
```

Or install the package locally and use the CLI:

```bash
pip install -e .
airca demo --output-dir output
```

## Why this matters

The goal is to make AIRCA immediately legible and usable.

A project like this only becomes adopted when people can try it without reading the full theory first.
